Rezter's New Way Theme Tutorial

How can I help?

  1. I want to change the title of my page
  2. I want to change the email address on my contact page
  3. I want to change the background image for my page
  4. I want to replace the header text with an image

I want to change the title of my page

  1. Locate this folder: phpRecDB/themes/NewWay/views/layouts
  2. Edit this file: _title.php
  3. Replace all the text in this document with what you want to appear as the title of your page.
  4. Save and upload the files to your server.

I want to change the email address on my contact page

  1. Locate this folder: phpRecDB/themes/NewWay/views/site/pages
  2. Edit this file: _email.php
  3. Replace all the text in this document with the email address you want to appear on your contact page.
  4. Save and upload the files to your server.

I want to change the background image for my page

  1. Find/create the image you would like to use as your background.
  2. Optimize the image for the web.
    1. If you want the image to fill the whole background I would suggest a resolution of 1920 x 1080
    2. To prevent the filesize becoming very large and taking longer to load, I would suggest saving the image as a JPEG at 30% quality
  3. Save/rename the image as the following: background.jpg
  4. Put the final image in the following folder: phpRecDB/themes/NewWay/images
    This should overwrite the background image.
  5. Save and upload the files to your server.

I want to replace the header text with an image

  1. Find/create the image you would like to use as your header.
  2. Optimize the image for the web.
    1. To prevent disturbing the navigation on the right, I would suggest setting the width to 470 pixels or less
    2. To prevent the filesize becoming very large and taking longer to load, I would suggest saving the image as a JPEG at 30% quality
  3. Save/rename the image as the following: header.jpg
  4. Put the final image in the following folder: phpRecDB/themes/NewWay/images
  5. Locate this folder: phpRecDB/themes/NewWay/views/layouts
  6. Edit this file: internal.php
  7. Find this section of code: <div id="header">
  8. Delete all the code between: <div id="header"> and the very next </div>
  9. After <div id="header"> enter the following code:

    <img src="phpRecDB/themes/NewWay/images/header.jpg" alt="" />
  10. Save and upload the files to your server.
